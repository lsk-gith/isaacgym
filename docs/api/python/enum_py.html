class isaacgym.gymapi.JointType

模拟器支持的关节类型

成员：

JOINT_INVALID : 无效/未知/未初始化的关节类型。

JOINT_FIXED : 固定关节。两个刚体将一起运动。

JOINT_REVOLUTE : 旋转关节或铰链关节。刚体将绕一个指定轴旋转。

JOINT_PRISMATIC : 棱柱关节。刚体将沿一个轴线性移动。

JOINT_BALL : 球关节。刚体将在参考点上绕所有方向旋转。

JOINT_PLANAR : 平面关节。刚体将在指定平面上移动。

JOINT_FLOATING : 浮动关节。刚体之间无约束。

class isaacgym.gymapi.DofDriveMode

用于控制执行器自由度的可能驱动模式。设置为特定驱动模式的自由度将忽略其他模式的驱动命令。

成员：

DOF_MODE_NONE : 自由度自由运动，没有任何控制。

DOF_MODE_POS : 自由度将响应位置目标命令。

DOF_MODE_VEL : 自由度将响应速度目标命令。

DOF_MODE_EFFORT : 自由度将响应力（或扭矩）命令。

class isaacgym.gymapi.MeshType

模拟器使用的网格类型

成员：

MESH_NONE

MESH_COLLISION : 碰撞网格。网格仅用于碰撞检查和惯性计算。为获得更好性能，应使用粗粒度的凸网格近似刚体体积。

MESH_VISUAL : 视觉网格。网格仅用于渲染。

MESH_VISUAL_AND_COLLISION : 视觉与碰撞网格。网格同时用于渲染和碰撞检查。

class isaacgym.gymapi.ImageType

传感器生成的图像类型

成员：

IMAGE_COLOR : RGB图像。标准相机传感器生成的图像。每个像素由所选数据类型（GymTensorDataType）的三个值组成，分别表示红、绿、蓝的强度。

IMAGE_DEPTH : 深度图像。每个像素是所选数据类型（GymTensorDataType）的一个值，表示该点到相机中心的距离。

IMAGE_SEGMENTATION : 分割图像。每个像素是所选数据类型（GymTensorDataType）的一个整数值，表示该像素上显示的物体类别。

IMAGE_OPTICAL_FLOW : 光流图像 - 每个像素是一个2D向量，表示该像素可见物体的屏幕空间速度。
