apply_actor_dof_efforts(self: Gym, arg0: Env, arg1: int, arg2: numpy.ndarray[float32]) → bool
Applies efforts passed as an ordered array to the Degrees of Freedom of an actor.（将按顺序数组传递的努力应用于 Actor 的自由度。）

  • If the Degree of Freedom is linear, the effort is a force in Newton.（如果自由度是线性的，则努力是牛顿力。）

  • If the Degree of Freedom is revolute, the effort is a torque in Nm.（如果自由度是旋转的，则努力是扭矩，单位为 Nm。）

See isaacgym.gymapi.Gym.set_dof_actuation_force_tensor for new tensor version of this API.（有关此 API 的新张量版本，请参见 isaacgym.gymapi.Gym.set_dof_actuation_force_tensor。）

Parameters:

  • param1 (Env) – Environment Handle（环境句柄）

  • param2 (Actor) – Actor Handle（Actor 句柄）

  • ( (param3) – obj: list of float) : array containing the efforts for all degrees of freedom of the actor.（包含 Actor 所有自由度努力的数组。）

Returns:

true if viewer has closed, false otherwise（如果查看器已关闭则为 true，否则为 false）

Return type:

bool
