class isaacgym.gymapi.AttractorProperties

The Attractor is used to pull a rigid body towards a pose. Each pose axis can be individually selected.（吸引器用于将刚体拉向一个姿态。每个姿态轴可单独选择。）

property axes

Axes to set the attractor, using GymTransformAxesFlags. Multiple axes can be selected using bitwise combination of each axis flag. if axis flag is set to zero, the attractor will be disabled and won’t impact in solver computational complexity.（使用 GymTransformAxesFlags 设置吸引器的轴。可以使用每个轴标志的按位组合选择多个轴。如果轴标志设置为零，吸引器将被禁用，不会影响求解器的计算复杂度。）

property damping

Damping to be used on attraction solver.（吸引求解器使用的阻尼。）

property offset

Offset from rigid body origin to set the attractor pose.（从刚体原点到吸引器姿态的偏移。）

property rigid_handle

Handle to the rigid body to set the attractor to（要设置吸引器的刚体句柄）

property stiffness

Stiffness to be used on attraction for solver. Stiffness value should be larger than the largest agent kinematic chain stifness（求解器吸引使用的刚度。刚度值应大于最大代理运动链的刚度）

property target

Target pose to attract to.（要吸引到的目标姿态。）

Collision filter bitmask - shapes A and B only collide if (filterA & filterB) == 0.（碰撞过滤器位掩码 - 只有当 (filterA & filterB) == 0 时，形状 A 和 B 才会碰撞。）

Coefficient of static friction. Value should be equal or greater than zero.（静摩擦系数。值应大于或等于零。）

property rest_offset

How far objects should come to rest from the surface of this body (used with PhysX only)（物体应该离此物体表面多远才能静止（仅用于 PhysX））

property restitution

Coefficient of restitution. It’s the ratio of the final to initial velocity after the rigid body collides. Range [0,1]（恢复系数。它是刚体碰撞后最终速度与初始速度之比。范围 [0,1]）

class isaacgym.gymapi.FlexParams

Simulation parameters used for FleX physics engine（用于 FleX 物理引擎的仿真参数）

property contact_regularization

Distance for soft bodies to maintain against ground planes（软体与地平面保持的距离）

Flag to activate deterministic simulation. Flex Newton solver only（激活确定性仿真的标志。仅用于 Flex Newton 求解器）

property dynamic_friction

Coefficient of friction used when colliding against shapes（与形状碰撞时使用的摩擦系数）

property friction_mode

Type of friction mode:（摩擦模式类型：）

  • 0 single friction dir, non-linear cone projection, but can't change direction during linear solve（0 单摩擦力方向，非线性锥投影，但无法在线性求解期间改变方向）

    • 1 two friction dirs, non-linear cone projection, can change direction during linear solve（1 两个摩擦力方向，非线性锥投影，可以在线性求解期间改变方向）

    • 2 same as above plus torsional (spinning) friction（2 与上述相同，加上扭转（旋转）摩擦）

property geometric_stiffness

Improves stability of joints by approximating the system Hessian（通过近似系统 Hessian 矩阵提高关节的稳定性）

property max_rigid_contacts

Max number of rigid body contacts（刚体接触的最大数量）

property max_soft_contacts

Max number of soft body contacts（软体接触的最大数量）

property num_inner_iterations

Number of inner loop iterations taken by the solver per simulation step. Is used only by Newton solver.（求解器每个仿真步长进行的内循环迭代次数。仅用于 Newton 求解器。）

property num_outer_iterations

Number of iterations taken by the solver per simulation step.（求解器每个仿真步长进行的迭代次数。）

property particle_friction

Coefficient of friction used when colliding particles（粒子碰撞时使用的摩擦系数）

property relaxation

Control the convergence rate of the parallel solver. Values greater than 1 may lead to instability.（控制并行求解器的收敛速度。大于 1 的值可能导致不稳定。）

property return_contacts

Read contact information back to CPU（将接触信息读回 CPU）

property shape_collision_distance

Distance for soft bodies to maintain against rigid bodies and ground plane（软体与刚体和地平面保持的距离）

property shape_collision_margin

Distance for rigid bodies at which contacts are generated（刚体生成接触的距离）

property solver_type

Type of solver used:（使用的求解器类型：）

  • 0 = XPBD (GPU)（0 = XPBD (GPU)）

  • 1 = Newton Jacobi (GPU)（1 = Newton Jacobi (GPU)）

  • 2 = Newton LDLT (CPU)（2 = Newton LDLT (CPU)）

  • 3 = Newton PCG (CPU)（3 = Newton PCG (CPU)）

  • 4 = Newton PCG (GPU)（4 = Newton PCG (GPU)）

  • 5 = Newton PCR (GPU)（5 = Newton PCR (GPU)）

  • 6 = Newton Gauss Seidel (CPU)（6 = Newton Gauss Seidel (CPU)）

  • 7 = Newton NNCG (GPU)（7 = Newton NNCG (GPU)）
